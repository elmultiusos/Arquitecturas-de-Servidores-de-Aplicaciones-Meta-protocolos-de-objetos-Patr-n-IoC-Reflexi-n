Endpoints Dinamicos (IoC Framework)

Los siguientes endpoints son manejados via reflexion con anotaciones @RestController y @GetMapping:

  • GET /hello - Saludo basico
  • GET /greeting - Saludo con parametro por defecto (World)
  • GET /greeting?name=Juan - Saludo con @RequestParam

Arquitectura

  • SimpleWebServer: Servidor HTTP que atiende conexiones en un puerto configurable.
  • MicroSpringBoot: Framework IoC que escanea el classpath y registra controladores con @RestController.
  • @GetMapping: Anotacion para mapear metodos a rutas HTTP GET.
  • @RequestParam: Anotacion para inyectar parametros de query string en los metodos.
